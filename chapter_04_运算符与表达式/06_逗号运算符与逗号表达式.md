# 06_逗号运算符与逗号表达式

C语言中，可用逗号运算符将多个表达式分隔开来。

```c
表达式1, 表达式2, …, 表达式n
```

逗号表达式又称为顺序求值运算符，其求解过程是：先求解`表达式1`，再求解`表达式2`，一直求解到`表达式n`。整个逗号表达式的值是`表达式n`的值。

```C
int result_1 = 2 + 5, 1 + 2, 5 + 7;
int result_2 = (2 + 5, 1 + 2, 5 + 7);
```

```c
// 05_comma_operator.c
#include <stdio.h>

int main() {
    // int result_1 = 2 + 5, 1 + 2, 5 + 7;
    int result_2 = (2 + 5, 1 + 2, 5 + 7);
    printf("%d", result_2);
    return 0;
}
```
