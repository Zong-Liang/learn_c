# 04_嵌套的结构体

结构体中的成员不仅可以是基本类型，也可以是结构体类型。例如，定义一个电脑结构体类型，其中的成员包括电源、机箱、显示器和内置零部件。其中，内置零部件又属于一个结构体类型，包括CPU、主板、显卡3个成员。电脑这样的结构体类型就是嵌套的结构体。

```c
struct Built_in_Components {
    char cpu[20];
    char mainboard[20];
    char gpu[20];
};

struct Computer {
    char power[20];
    char box[20];
    char displayer[20];
    struct Built_in_Components components;
};
```
